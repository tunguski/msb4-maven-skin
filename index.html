Msb4 Maven Skin

Apache Maven skin built on Bootstrap 4.0

Bootstrap themes from Bootswatch

Select a free theme for your website from an excellent gallery at Bootswatch. Out of the box support for these and other custom Bootstrap themes.

Page layouts

Write plain text in Markdown or APT, then set different layouts to your page sections.

Modern skin

Msb4 skin uses modern components from Bootstrap and other libraries, upgrades Maven generated site and provides further enhancements.

New Velocity tools

The skin adds a library of new Velocity tools to use in your own Maven template: rewrite HTML code, support per-page configurations and more!

Get it now

To use Msb4 skin in your Maven site, add it to site.xml:

<skin>
  <groupId>pl.matsuo.maven.skins</groupId>
  <artifactId>msb4-maven-skin</artifactId>
  <version>0.1.0-SNAPSHOT</version>
</skin>

Furthermore, the skin requires accompanying Msb4 Velocity tools as a dependency as well as Apache Velocity 1.7. Add them as dependencies to maven-site-plugin.

Responsive layouts

Write your pages in APT or Markdown, then restructure them using Msb4 skin layouts:

  • Carousel - spinning image slideshow
  • Thumbnails - showcase your image gallery
  • Columns - multi-column text
  • Sidebar - wrap into a sidebar
  • Body - text as it has been written

Partition the page into sections using <hr/> elements, and define preferred layouts for each section in site.xml. Msb4 skin is responsive thanks to Bootstrap, so the layouts will be rearranged automatically for readability on small screens.

Themes

The skin theme can be switched easily: just select a Bootstrap theme to give an easy makeover for your Maven site.

  • Default - use the default Bootstrap theme
  • Bootswatch - select an excellent free theme from Bootswatch
  • Custom - create your own Bootstrap theme with existing tools

Need to change something? Extend the skin with custom site.css file in your project, and reuse it for multi-module site.

Configure

Msb4 is very configurable: many features and components can be disabled, and optional enhancements enabled using configuration in site.xml:

  • Table of contents - display ToC for each page: top bar or sidebar
  • Menus - filter Maven menus and select what to display in top or bottom navigation
  • Code highlight - syntax colouring for code snippets
  • Image preview - display images in pop-ups
  • â¦ and more

Check out the documentation for all the features. Every configuration option can be applied on a per-page basis!

Velocity tools

Msb4 skin provides custom Velocity tools library to be used in Maven site template:

  • SkinConfigTool - convenient access to custom configuration options (global and per-page)
  • HtmlTool - query and modify HTML text
  • URITool - use Java URIs in the template

To enable these tools for any skin, add msb4-velocity-tools dependency to maven-site-plugin in the POM.

About

Msb4 Maven skin is an Apache Maven site skin built on Bootstrap 4.0.

The code is open source and licensed under Apache license. The skin can be used freely for your Maven projects.

Contribute by reporting issues, suggesting new features, or forking the Git repository on GitHub and adding some good code!

In the end, if you really really like the skin and want to support the author, I will be glad to accept a small donation.
