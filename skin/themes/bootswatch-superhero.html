Bootswatch Superhero theme

This page is using a free Superhero theme, courtesy of Bootswatch.

To use this theme with Msb4 Maven skin, use the following <theme> configuration in the site.xml site descriptor file:

<custom>
  <msb4Skin>
    ...
    <theme>bootswatch-superhero</theme>
    ...
  </msb4Skin>
</custom>
