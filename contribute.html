Contribute

Msb4 skin is an open-source project and all contributions are welcome to assist with its development and maintenance. The skin originated as a new skin for the upcoming Community Z Tools release, which uses Maven site for its documentation.

Along the way I realised that the skin is becoming general enough to be released separately so that the whole community can benefit. Msb4 Maven skin and the associated Msb4 Velocity tools can be used freely and are released under Apache license.

Issues (bug and feature tracker)

Please report any bugs found, feature requests or other issues on Msb4 skin GitHub tracker.

When creating a new issue, try following necolas’s guidelines.

Fork, patch and contribute code

Feel free to fork Msb4 skin’s Git repository at GitHub for your own use and updates.

Contribute your fixes and new features back to the main codebase using GitHub pull requests.

Support

Let me know if you are using Msb4 skin in your project. I may get around to creating a showcase page listing skin user sites.

Contact me if you have questions about the skin, or just like to say something about it. If you really really like the skin and want to support the author, I will be glad to accept a small donation.

Acknowledgements

Msb4 skin is built on Bootstrap and uses themes from Bootswatch. These include icons from Glyphicons and web fonts from Google.

JavaScript goodies with jQuery.

Image previews with Lightbox2.

Code highlighting with highlight.js.

HTML rewriting with jsoup.

Some artwork resources from MediaLoot.

Based on Reflow Maven Skin. Inspired by Maven Fluido Skin.
